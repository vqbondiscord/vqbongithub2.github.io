About TaskViewer

TaskViewer.org is a lightweight application that monitors everything the user has running. It features a small installation file, real-time updating, fast indexing and searching, and uses minimal system resources. Ideal for anyone needing full visibility into running processes.

Features

  • Small installation file
  • Clean and simple user interface
  • Quick file indexing
  • Quick searching
  • Quick startup
  • Minimal resource usage
  • Small database on disk
  • Real-time updating
